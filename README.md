# Pixijs - Zombie Game

## Description

末世生存类游戏，

1. 这个游戏是一个 2D， 用 pixijs 和 ts 写的游戏
2. 游戏的主要玩法是生存、种地类型，玩家需要在一个末世世界中生存下去
